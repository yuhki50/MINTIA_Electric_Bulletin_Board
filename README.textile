h1. MINTIA Electronic Bulletin Board

This is the small electric bulletin board which incorporating an LED matrix MINTIA case.
It can read and display the animated GIF file written in black and white MicroSD.

It can read GIF animation file (monochrome) from MicroSD, and display.
